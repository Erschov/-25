# GitHub User Finder
**Автор:** [Ваше Имя Фамилия]
## Описание
Приложение «GitHub User Finder» позволяет искать пользователей GitHub через API, просматривать их основную информацию и сохранять избранных пользователей в локальный JSON‑файл.
## Функциональность
* Поиск пользователей GitHub по имени.
* Отображение информации о пользователе (логин, имя, количество публичных репозиториев).
* Добавление пользователей в избранное.
* Удаление пользователей из избранного.
* Сохранение избранного в JSON‑файле.
## Использование API GitHub
Приложение использует GitHub API v3 для получения информации о пользователях. Базовый URL: `https://api.github.com/users/{username}`.
API возвращает JSON‑объект с данными пользователя. Приложение извлекает:
* `login` — логин пользователя.
* `name` — полное имя (может отсутствовать).
* `public_repos` — количество публичных репозиториев.
## Инструкция по запуску
1. Установите Python 3.6+.
2. Установите зависимости: `pip install requests`.
3. Запустите приложение: `python main.py
`.
4. Введите имя пользователя GitHub в поле ввода и нажмите «Найти».
5. Для добавления в избранное выберите пользователя из результатов и нажмите «Добавить в избранное».
6. Избранные пользователи отображаются в отдельном списке.
